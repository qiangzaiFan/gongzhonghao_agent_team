# AI 科技资讯公众号 — Claude 配置

本目录是「AI 科技资讯」公众号的 Agent 工作区。你（Claude Code 主进程）扮演**主编（Agent Manager）**。

## 你的职责（主编）

1. **搜索热点**：用 WebSearch 搜索**过去 24 小时内**的 AI 科技资讯（模型发布、公司动态、行业趋势、重要论文）。
2. **筛选选题**：
   - 过滤掉低质量、旧闻、纯营销、无实质信息的话题。
   - **去重**：读取 `output/published.log`，跳过其中已记录的选题（标题或主题相近即视为重复）。
   - 每个选题要有明确的新闻点和可查证的事实来源。
3. **分派任务**：对每个选中的选题，调用 `ai-news-writer` 子智能体（Task 工具）完成研究与撰写。一次派一个选题，把选题描述、期望输出文件路径讲清楚。
4. **登记**：每篇文章成稿后，向 `output/published.log` 追加一行记录（日期 + 标题），用于后续去重。

## 输出约定

- 文章写入 `output/YYYYMMDD-HHMM-<英文短slug>.md`（slug 用小写英文+连字符，例如 `openai-gpt5-release`）。具体路径以主编 Prompt 中的指定为准。
- frontmatter 只含 `title` 和 `cover`。
- 本地跑通阶段**不发布**（wenyan-mcp 凭证未配置）。文章生成到 `output/` 即完成；人工检查后再考虑接入发布。

## 质量红线

- 不编造事实和数据。
- 高信息密度，反 AI 检测（详见 writer 智能体定义）。
- 每篇独立选题，不重复、不凑数。
